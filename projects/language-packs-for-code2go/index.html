Code2Go is a code editor for Windows phone. Code2Go allows you to add highlighting and formatting support for new or existing language. This Github repository contains such language packs which I created. You can import these language packs into the Code2Go editor and use them. This will help you to use the code editor for various languages which it does not natively support.

{% include project-detail-links.html %}
Warning
This project is not being developed due to the availablity of better code editors on the platform.

Usage

  1. Copy the .cgi file to your phone.
  2. Open the Code2Go app. In the home screen you will see an option to Import package.
  3. Click on Import package button.
  4. Now browse the location of the .cgi file.
  5. Select that file and click open.
  6. The new language pack will be added to Code2Go.

Supported Languages

  • Java
  • Python
  • C++
  • HTML
  • Visual Basic
