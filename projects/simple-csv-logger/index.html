Simple CSV Logger

This is a C# logger class which logs error and other debug info into CSV files instead of a traditional text files. The logger also contains 6 levels of logging and log from text messages as well as exceptions.

Usage

The logger class creates CSV log files in the projects main directory. This directory is choosen so that the class can be used for both windows application and web based applciations. Both the filename and the diretory path are set at the constructor. Modify them to suit your application.
